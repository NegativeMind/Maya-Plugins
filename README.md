Maya-Plugins
============

学生の頃にC++の勉強として書いたAutodesk MayaのShaderプラグイン。
書いたのは確か2009年だった気がする。それまでC++もプラグイン開発もやったことがなかった。

ということで非常に非効率なコーディングとなっております。(staticの使い方を確実に間違えている…)

このShaderを使うとこんな質感になります↓
